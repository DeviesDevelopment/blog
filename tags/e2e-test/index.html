Why you should add a cypress test to your CI/CD pipeline

May 12, 2022
Cypress is a framework for testing web applications, primarily used for end-to-end testing. In an end-to-end test you test your application as a whole just as a real user would, by interacting with GUI components, without any mocked components. The goal of this post is to convince you to add a single simple cypress test to your CI/CD pipeline. In end-to-end tests with cypress you can test a lot of aspects of your application with very little test code.
